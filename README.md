# Git for Power BI specialists — інтерактивний курс

Багатосторінковий інтерактивний курс українською: від першої команди в терміналі до командної роботи з PBIP, конфліктів TMDL/JSON і деплою через deployment pipeline у Fabric.

**52 уроки · 10 модулів.** Покрокові SVG-графи історії, мультирепозиторні сценарії «двоє розробників», симулятор команд, впорядкування кроків, 34 міні-перевірки, банки вправ, довідник з 30 реальних проблем, глосарій 60+ команд. Прогрес зберігається в localStorage браузера. Пошук — по всьому курсу.

## Локальний перегляд

```bash
cd docs && python3 -m http.server 8000
# http://localhost:8000
```

## Публікація (GitHub Pages)

```bash
./publish.sh                # або ./publish.sh моя-назва-репо
```

Потрібен встановлений [GitHub CLI](https://cli.github.com/) і `gh auth login`. Скрипт створює публічний репозиторій, пушить код і вмикає Pages через Actions. Далі кожен push у `main` проходить QA-гейт і деплоїться автоматично.

## Розробка (goal workflow loop)

```bash
npm install
node scripts/build-search-index.mjs   # після зміни контенту
node scripts/qa.mjs                   # цілі G1–G6; exit 0 = готово
```

Специфікація циклу, цілі G1–G7, правила контенту і конвенції віджетів — у [CLAUDE.md](CLAUDE.md).
