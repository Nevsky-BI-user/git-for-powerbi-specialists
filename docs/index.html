Git для Power BI-спеціалістів

Від першої команди в терміналі до командної роботи з PBIP, вирішення конфліктів TMDL/JSON і публікації через Git sync робочої області (за потреби — deployment pipeline). З нуля, простою мовою, з інтерактивними схемами, симуляторами команд і міні-вправами під кожною темою.

Прогрес зберігається у твоєму браузері. Почни з модуля 00 або обери потрібний.
